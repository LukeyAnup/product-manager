# Product App (React)

A simple product listing application built with **React** that fetches products from an API and allows users to view product details and manage favorites.

---

## Features

- Product listing
- Product details page
- Favorites (stored in localStorage)
- API integration
- Responsive UI

---

## Tech Stack

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Zustand** (state management)
- **React Router**
- **Cypress** (testing)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/LukeyAnup/product-manager.git
```

Navigate to the project folder
cd your-repo-name

Install dependencies
npm install

Create a .env file
VITE_API_URL=https://your-url

Run the project
npm run dev

Run tests
npx cypress open

run json server
json-server --watch db.json --port 4000
